Package src.java.com.infusion.util.event

 

Interface Summary
EventBroker Iterface for defining an event broker - an object to which people subscribe to events and publish events.
EventConfiguration Provides a configuration for event types and consumers.
EventConsumer A class that will handle an event of a particular type.
EventDelegate A delegate is passed into the eventBroker as an event, but defers the creation of the event object until the broker knows that there are people that actually care about the event.
 

Class Summary
EventBrokerHolder Allows you to access the event broker staticly.
EventBrokerImpl The main service that deals with registering and locating consumers for events.
EventType An event type is a combination of the object class to be passed as the event as well as the name of the event as it was registered.
 

Exception Summary
EventException An exception thrown when dealing with events.
